# 1.3. Aspectos matemáticos de la graficación.
Para que una computadora dibuje un objeto y lo mueva en un espacio 3D, todo se reduce a cálculos matemáticos. No se trata solo de colores, sino de geometría pura. 
## 1. Espacios Vectoriales y Puntos.
En graficación, todo objeto está compuesto por vértices. Un vértice es simplemente un punto en un espacio coordenado.
* 2D: Se define por un par $(x, y)$.
* 3D: Se define por una terna $(x, y, z)$.Los vectores son fundamentales porque definen direcciones (como hacia dónde apunta una luz) o distancias entre puntos.

## 2. Transformaciones Geométricas (Matrices)

Esta es la parte más importante. Para mover un objeto, se utilizan matrices de transformación. En lugar de calcular cada punto por separado, se multiplica la matriz del objeto por una matriz de transformación.
* Traslación: Mover el objeto de un lugar a otro.Rotación: Girar el objeto sobre un eje. Se usan funciones trigonométricas ($\sin$, $\cos$).
* Escalamiento: Cambiar el tamaño del objeto.
Para que estas operaciones sean eficientes y se puedan combinar en un solo cálculo, se utilizan Coordenadas Homogéneas, que añaden una cuarta dimensión ($w$) para trabajar con matrices de $4 \times 4$.3.

## 3. Proyecciones: De 3D a 2DComo los monitores son planos (2D), necesitamos fórmulas para "aplastar" el mundo 3D sin que pierda sentido.
* Proyección Ortográfica: Mantiene las proporciones reales (se usa mucho en planos de ingeniería/CAD).
* Proyección de Perspectiva: Los objetos más lejanos se ven más pequeños, simulando cómo ve el ojo humano. Aquí interviene el concepto del punto de fuga.

## 4. Álgebra Lineal y el Producto Punto/Cruz
* Producto Punto ($\cdot$): Se usa para calcular el ángulo entre dos vectores. Es vital para la iluminación; si el ángulo entre la luz y la superficie es pequeño, el objeto brilla más.
* Producto Cruz ($\times$): Se usa para encontrar el vector normal (la dirección perpendicular a una cara). Sin esto, la computadora no sabría cuál es el "frente" o la "espalda" de un polígono.

## 5. Curvas y Superficies (Polinomios)
Para no dibujar todo con líneas rectas y planas, se usan matemáticas de curvas suaves:
* Curvas de Bézier y B-Splines: Basadas en polinomios, permiten crear formas curvas (como la carrocería de un auto en 3D o una fuente de texto) usando solo unos pocos puntos de control.
