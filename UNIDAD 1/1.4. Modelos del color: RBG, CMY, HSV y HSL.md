# 1.4. Modelos del color: RBG, CMY, HSV y HSL.
<p align="justify">
Para que una computadora represente colores, necesita un sistema numérico que le diga cuánta luz o cuánta tinta usar. Estos se dividen principalmente en modelos aditivos (luz) y sustractivos (pigmentos), o modelos basados en la percepción humana.

  ## 1. Modelo RGB (Red, Green, Blue)Es el modelo aditivo. 
Se basa en la mezcla de luces de color Rojo, Verde y Azul. Es el estándar para todo lo que tenga una pantalla (monitores, celulares, proyectores).
* Cómo funciona: Parte del negro (ausencia de luz). Al sumar los tres colores al máximo ($255, 255, 255$), se obtiene el blanco.
* Uso: Gráficos web, videojuegos y pantallas digitales.
## 2. Modelo CMY / CMYK (Cyan, Magenta, Yellow)
Es el modelo sustractivo. Se basa en la forma en que los pigmentos absorben la luz. Se usa exclusivamente para medios físicos (papel).
* Cómo funciona: Parte del blanco (el papel). Al mezclar Cian, Magenta y Amarillo, se obtiene un tono oscuro (teóricamente negro). En la práctica, se añade el Key (Negro/K) para ahorrar tinta y dar profundidad.
* Uso: Imprentas, revistas y folletos.

## 3. Modelos HSV y HSL (Basados en la percepción)
Estos modelos no piensan en "mezclas de botes", sino en cómo los humanos describimos el color. Ambos se representan comúnmente en un cilindro o cono.

### HSV (Hue, Saturation, Value)
* Hue (Tono): El color puro (rojo, azul, etc.), medido en grados de 0° a 360°.
* Saturation (Saturación): Qué tan "gris" o qué tan "vivo" es el color.
* Value (Valor): El brillo o la cantidad de luz (de negro a color puro).

### HSL (Hue, Saturation, Lightness)
Es muy parecido al HSV, pero cambia la forma en que maneja la iluminación:
* Lightness (Luminosidad): A diferencia del "Value", aquí el 100% de luminosidad siempre es blanco puro, mientras que el 50% es el color original.
* Uso: Es el favorito de los diseñadores web y programadores de CSS porque es más fácil de ajustar visualmente que el RGB.</p>
